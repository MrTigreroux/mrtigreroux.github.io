Welcome to the messages updater tool for {{ name }} plugin.
This tool allows you to update your messages to the latest version of {{ name }}.
However, it is recommended to have a look at the update logs (before updating the plugin) to see the messages that have changed.

Bienvenue sur l'outil de mise à jour des messages pour le plugin {{ name }}.
Cet outil vous permet de mettre à jour vos messages vers la dernière version de {{ name }} sans avoir à prendre ceux de base (qui sont disponibles sur la page de présentation du plugin).
Cependant, il est recommandé de lire le détail des mises à jour (avant de mettre à jour le plugin) pour voir les messages qui ont changé.

Warning

This tool doesn't add the missing placeholders ("_Placeholder_").
It doesn't fix syntax errors, accents, or all other problem already present in the given messages.
The (old) unused messages are removed and displayed at the bottom of the page.

Attention

Cet outil ne rajoute pas les placeholders ("_Placeholder_") manquants.
Il ne corrige pas non plus les erreurs de syntaxe, les accents, ou tout autre problème présent dans les messages fournis.
Les (anciens) messages non utilisés sont retirés et indiqués en bas de page.
